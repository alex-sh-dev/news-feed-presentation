# Лента новостей

Данный проект был выполнен в рамках тестового задания. Необходимо было разработать приложение, отображающее
новостную ленту компании

Для получения данных использовались следующие API функции
- webapi.autodoc.ru/api/news/{page}/{count} - список новостей, где {page} - это номер страницы, а {count} - это количество элементов на ней (например, webapi.autodoc.ru/api/news/1/15)
- webapi.autodoc.ru/api/news/item/{url} - данные конретной новости, где {url} - это атрибут элемента списка новостей (например, webapi.autodoc.ru/api/news/item/avto-novosti/volvo_ex30_base)

При выполнении задания требовалось придерживаться этого стека технологий MVVM + Combine, UICollectionView + CompositionalLayout + DiffableDataSource

Другие требования:
- Загрузка и отображение постранично
- Все запросы async/await
- Оптимизация для iPad
- Без использования сторонних библиотек

Для ознакомления с вариантом кода, который был предоставлен проверяющей стороне см. [ветку exam](https://github.com/alex-sh-dev/news-feed-presentation/tree/exam) (содержит Сообщение разработчику в Readme)

Была получена обратная связь и планируется выполнить работу над ошибками по возможности
